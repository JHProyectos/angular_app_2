Guía de Despliegue en GitHub Pages

Instrucciones paso a paso para desplegar tu aplicación Angular

1

Crear repositorio en GitHub

Crea un repositorio vacío en GitHub (sin README ni .gitignore)

2

Clonar el repositorio

cd ruta/a/tu/directorio
  git clone https://github.com/JHProyectos/nombre-del-repo.git
  cd nombre-del-repo
3

Crear proyecto Angular

ng new nombre-del-repo
  cd nombre-del-repo
4

Configurar y probar la app

# Modifica src/app/ según necesites
  ng serve  # Probar en http://localhost:4200
5

Hacer el primer push

git add .
  git commit -m "Inicializar proyecto Angular"
  git push origin main
6

Agregar angular-cli-ghpages

ng add angular-cli-ghpages
7

Construir la app

ng build --output-path dist/nombre-del-repo --base-href /nombre-del-repo/
8

Desplegar

Primero verificar ubicación de index.html

# Revisa la carpeta generada:
  dir dist\nombre-del-repo
  # Si index.html está en dist/nombre-del-repo, usa esa carpeta
  # Si ves una carpeta browser, y si index.html está en 
  dist/nombre-del-repo/browser, usa esa carpeta

Ahora usar npx para desplegar

# Si index.html está en dist/nombre-del-repo:
  npx angular-cli-ghpages --dir=dist/nombre-del-repo
  # Si index.html está en dist/nombre-del-repo/browser:
  npx angular-cli-ghpages --dir=dist/nombre-del-repo/browser
9

Configurar GitHub Pages

En GitHub: Settings > Pages > Source: gh-pages, / (root)

URL: https://JHProyectos.github.io/nombre-del-repo/

+

Actualizar la app después de modificarla

Sigue estos pasos para actualizar tu aplicación después de realizar cambios:

1. Editar y probar

# Edita src/app/ y prueba con:
  ng serve  # Probar en http://localhost:4200

2. Push al código fuente (Opcional)

git add .
  git commit -m "Actualizar proyecto con nuevos cambios"
  git push origin main

3. Reconstruir (requerido para reflejar cambios)

ng build --output-path dist/nombre-del-repo --base-href /nombre-del-repo/

4. Verificar index.html

dir dist\nombre-del-repo
  # Si ves una carpeta browser:
  dir dist\nombre-del-repo\browser

5. Redeployar con npx

# Si index.html está en dist/nombre-del-repo:
  npx angular-cli-ghpages --dir=dist/nombre-del-repo
  # Si index.html está en dist/nombre-del-repo/browser:
  npx angular-cli-ghpages --dir=dist/nombre-del-repo/browser

6. Revisar el sitio

Visita https://JHProyectos.github.io/nombre-del-repo/ para ver los cambios
